# Overview

I created a console app of a to do list that allows a user to add or remove items to a list, mark them as complete, and view the entire list. I created this project in order to learn more about the syntax of c#.

[Software Demo Video](https://youtu.be/VCWhSRQHCUw)

# Development Environment

I used C# with Visual Studio Code as my IDE with .NET SDK
